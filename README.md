# glg-add-to-consult

Polymer component that displays a modal dialog for adding an expert to a consultation selected by the user.

## Developing

1. clone this repo
1. `npm install -g git://github.com/custom-elements/polymer-serve#master`
1. `npm install -g livereload`
1. `npm install`
1. `bower install`

## Running demo

`npm test`
